Arbori binari de căutare

Un arbore binar de căutare (BST) este un arbore binar în care fiecare nod respectă următoarele condiții:
  • Cheile nodurilor din subarborele stâng sunt mai mici decât cheia nodului curent.
  • Cheile nodurilor din subarborele drept sunt mai mari decât cheia nodului curent.

Operații

  • Inserare: Adaugă un nou nod în poziția corespunzătoare păstrând proprietatea BST.
  • Căutare: Găsește un nod cu o cheie specificată.
  • Ștergere: Elimină un nod din arbore, ajustând structura pentru a păstra proprietatea BST.
  • Parcurgere: În inordine, preordine și postordine.

Inserare

void Inserare(nod* &R, int x) {
 if (R != NULL) {
     if (R->info == x) return;
     else if (R->info > x) Inserare(R->st, x);
     else Inserare(R->dr, x);
 } else {
     R = new nod{x, NULL, NULL};
 }
}
