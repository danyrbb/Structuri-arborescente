Arbori binari

Un arbore binar este o structură de date arborescentă în care fiecare nod are cel mult doi descendenți, denumiți stâng (left) și drept (right). Fiecare nod poate conține o valoare sau o cheie, și referințe către descendenții săi.

Proprietăți

Fiecare nod are 0, 1 sau 2 copii. Înălțimea arborelui este numărul maxim de muchii de la rădăcină până la o frunză.

Reprezentare

struct nod {
    int info;
    nod *st, *dr;
};

Arbori binari de căutare

Un arbore binar de căutare (BST) este un arbore binar în care fiecare nod respectă următoarele condiții:
  • Cheile nodurilor din subarborele stâng sunt mai mici decât cheia nodului curent.
  • Cheile nodurilor din subarborele drept sunt mai mari decât cheia nodului curent.

Operații

  • Inserare: Adaugă un nou nod în poziția corespunzătoare păstrând proprietatea BST.
  • Căutare: Găsește un nod cu o cheie specificată.
  • Ștergere: Elimină un nod din arbore, ajustând structura pentru a păstra proprietatea BST.
  • Parcurgere: În inordine, preordine și postordine.

Inserare

void Inserare(nod* &R, int x) {
    if (R != NULL) {
        if (R->info == x) return;
        else if (R->info > x) Inserare(R->st, x);
        else Inserare(R->dr, x);
    } else {
        R = new nod{x, NULL, NULL};
    }
}

Arbore binar complet

Un arbore binar complet este un arbore binar în care toate nivelurile, cu excepția posibilă a ultimului, sunt complet umplute, iar toate nodurile sunt cât mai la stânga posibil.

Proprietăți

Înălțimea arborelui este aproximativ log2(n). Este utilizat frecvent în structuri de date cum ar fi heap-uri.

Reprezentare

struct nod {
    int info;
    nod *st, *dr;
};
