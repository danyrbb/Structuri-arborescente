Arbori binari

Definiție și structură

Un arbore binar este o structură de date compusă din noduri, unde fiecare nod are cel mult doi copii, denumiți "stâng" și "drept". Componentele principale sunt nodurile, rădăcina, frunzele și înălțimea.

Operațiuni fundamentale

Traversările sunt metodele prin care parcurgem nodurile unui arbore. Există trei tipuri principale de traversări:

  • Preordine: Vizităm nodul curent înainte de subarborii săi.
  • Inordine: Vizităm nodul curent între vizitările subarborilor săi stâng și drept.
  • Postordine: Vizităm nodul curent după vizitarea subarborilor săi.

Reprezentare în cod

#include <iostream> using namespace std; void Preordine(int x) { if(x != 0) { cout << x << " "; Preordine(st[x]); Preordine(dr[x]); } } void Inordine(int x) { if(x != 0) { Inordine(st[x]); cout << x << " "; Inordine(dr[x]); } } void Postordine(int x) { if(x != 0) { Postordine(st[x]); Postordine(dr[x]); cout << x << " "; } }
